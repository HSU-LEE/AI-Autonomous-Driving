# IROC_AI_CAR


## 구성

- **AI 주행 코드**: `AI_CAR/`
  - 라즈베리파이에 올려서 실행하는 메인 폴더는 `AI_CAR/S1_main/` 입니다.
- **3D/기구 자료**
  - `f3d/`: Fusion 360 등 원본
  - `stl/`: 출력용 STL
  - `step/`: STEP 포맷
  - `photo/`: 사진/레퍼런스
  - 루트의 `*.pdf`: 바디/카메라 등 블루프린트 도면
